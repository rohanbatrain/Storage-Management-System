# Usage Guide

## Core Concepts

### Locations
Everything in SMS lives in a **Location**. Locations are hierarchical.
- **Example**: `Home` -> `Bedroom` -> `Wardrobe` -> `Top Shelf` -> `Blue Box`.
- **Root Locations**: Start by creating high-level locations like "Home" or "Office".

### Items
Items are the things you track.
- **Generic Items**: Electronics, tools, documents, etc.
- **Clothing Items**: Have extra metadata like brand, size, color, material.

## Features

### üì¶ Moving Items
1.  Navigate to an item's detail page.
2.  Click **"Move to..."**.
3.  Select the destination location from the tree.
4.  Optionally add a note (e.g., "Moved for cleaning").
5.  Click **Confirm Move**.

### ü§ù Lend to Friend
Track items you lend to others so you don't lose them!
1.  Go to an item's detail page.
2.  Click **"Lend to Friend"**.
3.  Enter the borrower's name and an optional due date.
4.  The item will be marked as "Lent" and appear in the **Lent Items** page (sidebar).
5.  When returned, go to the item or Lent Items page and click **"Mark Returned"**.

### ‚ö†Ô∏è Lost & Found
If you can't find an item:
1.  Go to the item's detail page.
2.  Click **"Mark as Lost"**.
3.  Add a note about where it was last seen.
4.  The item will be marked as "Lost" (red banner) and appear in the **Lost Items** page.
5.  When found, simply click **"Mark Found"** to restore it to its last known location.

### üëï Wardrobe & Laundry
- **Outfits**: Create outfits by selecting items from your wardrobe.
- **Laundry Flow**:
    - **Wear**: Mark an item as "Worn" (adds to history).
    - **Hamper**: Move worn items to "Laundry Hamper".
    - **Wash**: Process items in the Laundry view.
    - **Put Away**: Return clean items to their permanent storage location.

### üì∑ QR Codes
- Every item and location has a unique ID.
- Use the **Print QR** page to generate labels for your boxes and items.
- Scan a QR code (using the mobile app or a scanner) to instantly open that item/location.
